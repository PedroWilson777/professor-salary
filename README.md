# ProfPay — Calculadora de Salário de Professor

Sistema web para gerenciar professores, montar grade horária semanal e calcular pagamentos por hora-aula.

## Rodar localmente
```bash
npm install
npm run dev
```

## Deploy Vercel
```bash
npx vercel --prod
```

## API
`POST /api/salary/calculate` - calcula pagamento dos professores
